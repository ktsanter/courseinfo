I encourage you to retry programming assignments. It can take a few tries to master some of the concepts in this course and practicing in code is the best way to get there.
You cannot, however, retake tests or quizzes.
The best way by far to learn this material is by actually coding. Whenever you are given a new concept you should try it out in code, even if there's no assignment for it. Writing many small programs, experimenting, and researching on the Internet are the best ways to master programming. Contact me if you're looking for ideas about practice programs.
There is no final exam but there is a final project. You can find it in the Application and Review unit.
